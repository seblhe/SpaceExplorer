# SPACE EXPLORER

## lancer le jeu

### en direct

Avec Python 3 :

`python -m http.server 8080`


Avec Node.js :

`npx http-server -p 8080`


Puis ouvre ton navigateur à l’adresse :

[Localhost](http://localhost:8080/)

### via npm

Avec npm :

`npm run dev`

Puis ouvre ton navigateur à l’adresse :

[Localhost](http://localhost:5173/)
